## BLOQUE 1 — Buscador con filtros avanzados

**Objetivo:** Añadir checkboxes y un desplegable de plataformas al buscador actual.

**Tareas concretas:**
- Añadir checkbox "Mostrar DLCs" (por defecto: ocultos). Filtrar usando el campo category de IGDB (DLC = 1, expansión = 2, juego base = 0)
- Añadir checkbox "Mostrar Expansiones" (misma lógica, category = 2)
- Añadir desplegable de Plataformas: hacer una llamada al endpoint /platforms de IGDB para obtener la lista dinámica. Al seleccionar una plataforma, añadir where platforms = [id] a la query de búsqueda
- El estado de los filtros debe persistir mientras el usuario navega dentro del buscador

**Estado:** Incompleto.

**Problemas**: 
- En el desplegable de plataformas no aparece ninguna plataforma
- El buscador ahora no funciona

**Posibles soluciones:**
- Podriamos hacer una query al endpoint ```platforms```y así rellenar ese dropdown

## BLOQUE 3 — Tiempo para completar un juego

**Objetivo:** Mostrar cuánto tarda en completarse un juego.

**Endpoint:** POST https://api.igdb.com/v4/game_time_to_beats

```
fields normally, completely, hastily, game_id;
where game_id = [ID_DEL_JUEGO];

hastily = speedrun / mínimo
normally = historia principal
completely = 100%
```

**Mostrar en:** Página de detalle del juego (y opcionalmente en los tiles de Library)

**Formato sugerido:** "🕐 Historia: 12h · Completo: 30h"

**Estado:** Completado
