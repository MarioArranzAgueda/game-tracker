# Plan de Acción

## Contexto del proyecto

- Stack: React + TypeScript + Tailwind
- API principal: IGDB (https://api.igdb.com/v4/)
- Base de datos: Postgresql + Prisma
- Estado global: Tanstack-query

## BLOQUE 1 — Home page - Botón Filtros

**Objetivo:** Dar vida al botón de filtros

**Tareas concretas:**
- Añadir una animación que muestre los filtros
- Cambiar el diseño de los filtros por algo más bonito
- Si abro los filtros y navego a una pagina y vuelvo, los filtros deben de seguir viendose

**Estado:** Completo.

## BLOQUE 2 — Home page - Drag and drop
**Objetivo:** Utilizar una libreria

**Tarea:**
- Utilizar una libreria que realice el drag and drop
- No es muy visual y hay que mejorar el comportamiento
- Debe de ser accesible
- Debe de funciona en movil correctamente
- Añadir un numero en la portada del juego, para saber el orden que tiene
- Sobre ese numero, añadir un color para identificarlo

**Estado:** Completo

## BLOQUE 3 — Details - Información

**Objetivo:** Los textos se cortan

**Tarea:**
- Cuando estamos en movil, los textos se cortan y no se ve correctamente las plataformas, etc

**Estado:** Completo

## BLOQUE 4 — Eliminar juego

**Objetivo:** El alert del eliminar juego

**Tarea:**
- Eliminar el alert que aparece cuando eliminamos un juego
- Añadir un modal con el mismo texto y el mismo boton de confirmación
- Comprobar que en resolucion movil tambien funcinoa

**Estado:** Completo

## BLOQUE 5 — Eliminar juego

**Objetivo:** Poder borrar juegos sin entrar al detalle

**Tarea:**
- Se debe de añadir un boton que solo este en los juegos que esten añadidos
- Ese boton debe permitir borrar el juego directamente

**Estado:** Completo

## BLOQUE 6 — Boton añadir

**Objetivo:** El desplegable del boton añadir se debe cerrar

**Tarea:**
- Cuando queremos añadir un juego directamente desde la card, se abre un desplegable
- Si pulsamos fuera del desplegable se debe de cerrar
- Si pulsamos una de las opciones, se debe de cerrar
- Esta opción de añadir no se ve en movil ya que no hay estado hover

**Estado:** Completo
